🚀 راه‌اندازی

در کمتر از ۵ دقیقه

💻 محلی (توسعه)

۱. نصب پیش‌نیازها
pip install -r requirements.txt
۲. تنظیم محیط
cp .env.example .env
# ویرایش .env با اطلاعات ربات
۳. اجرا
python -m telegram_bot.main

☁️ Render (تولید)

۱

پروژه را روی GitHub قرار دهید

۲

در render.com یک Web Service جدید بسازید

۳

Build Command:

pip install -r requirements.txt
۴

Start Command:

python -m telegram_bot.main
۵

متغیرهای محیطی را اضافه کنید (BOT_TOKEN, CHANNEL_ID, ADMIN_IDS, BOT_MODE=webhook)

⚙️ متغیرهای محیطی (.env)

متغیر مقدار نمونه توضیح اجباری
BOT_TOKEN 123:ABCxyz... توکن ربات از BotFather ✓ بله
CHANNEL_ID @mychannel آیدی کانال برای انتشار نتایج ✓ بله
ADMIN_IDS 123456789 آیدی ادمین‌ها (با کاما جدا) ✓ بله
BOT_MODE webhook polling (محلی) یا webhook (Render) ✓ نه
WEBHOOK_HOST https://app.onrender.com آدرس Render در حالت webhook ✓ نه
DEFAULT_ROUNDS 5 تعداد دور پیش‌فرض هر مسابقه ✓ نه
